### strict mode     
Put this code on the top of your code ```'use strict;'```, no error would be found even if the browser does not support stict mode.

1. A variable must be declared with ```var```       
2. 


Example:        
```html
<html>
<body>
    <script type="text/javascript">
        'use strict'
        var name = 'Gabriel';
    </script>
</body>
</html>
```
