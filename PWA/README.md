# PWA Sample App

## Configuration

In manifest.json

-   display: How do you present your PWA in mobile phone
    -   `standalone`: To make your PWA looks like a native app.
    -   `browser`: Open your PWA in browser
-   background_color:
-   theme_color:
-   orientation: To launch the PWA in what screen orientation
